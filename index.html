class attributer

Hvis vi skal bruge gentagne styling på forskellige elementer, kan vi med fordel benytte class attributen. Fx ønsker vi at style nogle specifikke buttons, som skal have samme styling, kan vi med fordel benytte class attribute.

id attributer

id attributen gives til elementer, og skal være unikke. Det kan fx bruges på af css til styling, men også af javascript, hvis vi ønsker at der udføres en handling på et specifikt element.
